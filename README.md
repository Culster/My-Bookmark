# My-Bookmark
Apps to store knowledge.

---

This app is, at its core, for me who have a hobby to write a story, so I tend to save my references, and I have a LOT of it, so much so that sometimes I have a hard time searching for it again. Yes, even with browser's bookmarking system.
And so, I build this for myself.

---

- [ ] Notes Writing/Editing
- [ ] Bookmarks Viewing
- [ ] Bookmarks Writing/Editing
